## What Are Musical Constraints?

A **constraint** is a rule that limits what you *can* do — so you can focus on what you *should* do.

Think of it like a game. Basketball has rules: dribble the ball, no traveling, shoot through the hoop. Those rules don't make basketball *worse* — they make it a *game*. Without rules, it's just throwing a ball around.

Music works the same way. Every song you love is built on invisible rules — restrictions on which notes to use, how fast to play them, where the melody has to land. When you *choose* your rules instead of stumbling into them, you make better music.

In this workbook, you'll learn three types of musical constraints:

| Constraint | What It Does | Music Example |
|------------|-------------|---------------|
| **SNAP** | Restrict to a smaller set of notes | "Only use the white keys on the piano" |
| **FUNNEL** | Pull the melody toward a destination | "Start and end on C" |
| **TEMPO** | Limit which rhythm values you can use | "Only quarter notes and eighth notes" |

The big idea: **Constraints don't limit creativity — they channel it.**

Let's get started.

## Worksheet 1: SNAP — Pitch Subset Constraints

**What is SNAP?**
A SNAP constraint restricts you to a subset of pitches. Instead of all 12 notes in an octave, you might use only 7 (a scale), or 5 (a pentatonic scale), or even 3.

**Why it works:** Fewer notes means stronger identity. When you hear a melody that only uses C, E, and G, your ear recognizes it as *belonging together* — because those notes form a C major triad. The constraint creates coherence.

## Worksheet 2: FUNNEL — Tonal Gravity Constraints

**What is FUNNEL?**
A FUNNEL constraint says "the melody should move toward this note." It's like gravity — the tonic (home note) pulls the melody toward it. Most great melodies start near the tonic and end on it.

**Why it works:** Without a target, a melody wanders. With a funnel, every note has a sense of purpose — it's either heading home, arriving home, or leaving home to create tension before returning.

### Exercise 2A: Landing on C

**Constraint:** Your melody MUST end on the note C.
**Pitches:** Any note in C major (C, D, E, F, G, A, B — white keys only).
**Length:** 4 bars.

Write a melody that ends on C. Try to make the last note feel like "home."

```
Bar 1:  |____|____|____|____|
Bar 2:  |____|____|____|____|
Bar 3:  |____|____|____|____|
Bar 4:  |____|____|____|  C |
```

**Listen back:** Does the C at the end feel like an arrival? If it feels like the melody could continue, try adjusting the notes leading up to it. Steps (C→D, D→E) pull toward C more convincingly than leaps (G→C).

### Exercise 2B: The Round Trip

**Constraint:** Your melody must START on C AND END on C.
**Pitches:** White keys only (C major scale).
**Length:** 8 bars.

Think of it like a journey: you leave home, go exploring, and come back.

```
Bar 1:  | C  |____|____|____|   Bar 5:  |____|____|____|____|
Bar 2:  |____|____|____|____|   Bar 6:  |____|____|____|____|
Bar 3:  |____|____|____|____|   Bar 7:  |____|____|____|____|
Bar 4:  |____|____|____|____|   Bar 8:  |____|____|____| C  |
```

**Hint:** The notes right before the final C matter a lot. Try D→C (step) or B→C (half step). Both create strong pull toward C.

### Exercise 2C: The Waypoint

**Constraint:**
- Start on C
- Pass through G (the dominant) somewhere in bars 3–4
- End on C

**Pitches:** C major scale (white keys).
**Length:** 8 bars.

G is the "second most important note" in C major. Passing through it creates a sense of tension — you've gone up to the dominant, and now you need to find your way back down.

```
Bar 1:  | C  |____|____|____|   Bar 5:  |____|____|____|____|
Bar 2:  |____|____|____|____|   Bar 6:  |____|____|____|____|
Bar 3:  |____|__G_|____|____|   Bar 7:  |____|____|____|____|
Bar 4:  |____|____|____|____|   Bar 8:  |____|____|____| C  |
```

(G can go anywhere in bars 3–4 — the placement above is just one option.)

## Worksheet 3: TEMPO — Rhythmic Constraints

**What is TEMPO?**
A TEMPO constraint restricts which rhythm values you can use. Instead of every possible subdivision (whole, half, quarter, eighth, sixteenth, triplet...), you might use only quarters, or only quarters and eighths.

**Why it works:** Rhythmic constraints create groove. Most pop, rock, and classical music uses surprisingly few rhythm types. Too many rhythm values = chaos. A focused palette = momentum.

### Exercise 3A: Quarter Notes Only

**Constraint:** Use ONLY quarter notes (♩). Every note is exactly 1 beat.
**Pitches:** Free choice.
**Length:** 4 bars in 4/4 time.

```
Bar 1:  | ♩  | ♩  | ♩  | ♩  |
        |____|____|____|____|
Bar 2:  | ♩  | ♩  | ♩  | ♩  |
        |____|____|____|____|
Bar 3:  | ♩  | ♩  | ♩  | ♩  |
        |____|____|____|____|
Bar 4:  | ♩  | ♩  | ♩  | ♩  |
        |____|____|____|____|
```

Write a pitch under each quarter note. How much variety can you create when the rhythm never changes?

**Reflection:** With only one rhythm value, the *pitch* choices carry all the musical interest. Does this feel limiting or freeing? Why?

### Exercise 3B: Adding Eighths

**Constraint:** Use ONLY quarter notes (♩) and eighth-note pairs (♪♪).
**Pitches:** Free choice.
**Length:** 4 bars in 4/4 time.

Remember: eighth notes come in pairs. Each pair takes up 1 beat (♪♪ = 1 beat).

```
Bar 1:  |____|____|____|____|
Bar 2:  |____|____|____|____|
Bar 3:  |____|____|____|____|
Bar 4:  |____|____|____|____|
```

For each beat, write either 1 note name (quarter) or 2 note names separated by commas (eighth pair).

## Worksheet 4: Combining Constraints

**What happens when constraints stack?**

Each constraint alone shapes your music. Combine them, and they *interact* — the SNAP limits your pitches, the FUNNEL gives you a destination, and the TEMPO defines your rhythmic world. Together, they create a focused creative space where surprising things happen.

### Exercise 4A: The Full Stack (Tight)

**Constraints:**
- **SNAP:** Only C, E, G (3 pitches)
- **FUNNEL:** Start and end on C
- **TEMPO:** Only quarter notes and eighth notes

**Length:** 4 bars in 4/4.

```
Bar 4:  |____|____|____| C  |
```

## Composition Challenges

### Challenge 1: The Minimalist (Beginner)

**Constraints:**
- SNAP: Only 3 pitches of your choice
- FUNNEL: Start and end on the same note
- TEMPO: Only one rhythm value (your choice: all quarters, or all eighths, or all halves)

**Length:** 8 bars

**Goal:** Create something memorable using as little material as possible.

**Self-assessment:**
- [ ] Every note follows the SNAP constraint
- [ ] First and last notes are the same
- [ ] Every note is the same rhythm value
- [ ] I can hum it from memory after playing it once

---

### Challenge 2: The Pop Song (Intermediate)

**Constraints:**
- SNAP: Only the 5 notes of a major pentatonic scale (your choice of key)
- FUNNEL: Melody must end on the tonic (first note of the scale)
- TEMPO: Quarter notes and eighth notes only
- **Bonus constraint:** Include a repeating 2-bar pattern (A-A form)

**Length:** 8 bars (4 bars repeated)

**Goal:** Write something that could be a pop hook.

**Self-assessment:**
- [ ] Only pentatonic pitches used
- [ ] Ends on the tonic
- [ ] Quarter and eighth notes only
- [ ] Bars 1–4 = Bars 5–8 (exact repeat)
- [ ] It's catchy

- INTERVAL: No interval larger than a 4th (5 semitones)

**Length:** 8 bars

## Glossary of Terms

| Term | Definition |
|------|-----------|
| **Bar (Measure)** | A segment of time containing a specific number of beats. In 4/4 time, each bar has 4 beats. |
| **Beat** | The basic unit of time in music. In 4/4 time, there are 4 beats per bar. |
| **Chord** | Three or more notes played at the same time. |
| **Consonance** | A combination of notes that sounds stable and pleasant (e.g., a major triad). |
| **Constraint** | A rule that restricts musical choices. In this course: SNAP, FUNNEL, or TEMPO. |
| **Diatonic** | Using only the 7 notes of a major or minor scale (e.g., "white keys only" in C major). |
| **Dominant** | The 5th note of a scale (G in C major). Creates tension that wants to resolve to the tonic. |
| **Eighth note (♪)** | A note lasting half a beat. Two eighth notes fill one beat. |
| **FUNNEL** | A constraint that pulls the melody toward a destination note (like gravity). |
| **Half note** | A note lasting 2 beats. |
| **Interval** | The distance between two pitches, measured in semitones or scale degrees. |
| **Melody** | A sequence of single notes that forms a recognizable musical line. |
| **Pentatonic scale** | A 5-note scale. The C major pentatonic is C, D, E, G, A. Common in folk and pop music worldwide. |
| **Pitch** | How high or low a note sounds. C4 is middle C. |
| **Quarter note (♩)** | A note lasting 1 beat. The most basic rhythm value. |
| **Resolution** | The movement from tension (dissonance) to rest (consonance). |
| **Rhythm** | The pattern of long and short notes in time. |
| **Scale** | An ordered set of pitches, usually spanning one octave. |
| **Semitone** | The smallest interval in Western music. The distance between adjacent piano keys. |
| **SNAP** | A constraint that restricts you to a subset of pitches. "Snap" to a smaller set. |
| **TEMPO** | A constraint that restricts which rhythm values you can use. Also: the speed of music (measured in BPM). |
| **Tightness** | How strict a constraint is. Tight = very few choices. Loose = many choices but still limited. |
| **Tonic** | The "home" note of a key. In C major, the tonic is C. |
| **Triad** | A three-note chord (e.g., C major = C + E + G). |
| **Voice leading** | How individual voices (melody lines) move from one chord to the next. Smooth = small movements. |
